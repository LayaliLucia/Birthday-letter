# 🎉 Happy Birthday Website

I created this website as a **birthday gift for a friend**.  
It’s a remix of the **Valentines-Day** project by [jungcookgf](https://github.com/jungcookgf/valentines-day), with just a tiny modification to turn it into a cheerful birthday celebration.  

## ✨ Features
- Festive birthday theme 🎂
- Lighthearted animations and design
- Adapted from jungcookgf’s Valentines-Day project
- Personalized as a gift

## 🎯 Purpose
The goal of this remix was to:
- Learn by modifying existing code
- Practice customizing a project to fit a new occasion
- Share something joyful and personal with a friend

## 📌 Credits
All credits go to **jungcookgf** for the original Valentines-Day project.  
I only made small changes to adapt it for a birthday theme.
